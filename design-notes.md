# Ultra Premium Design Reference Notes

## SDAIA Color System
- Primary: #273470 (dark navy) / #3DB1AC (teal in dark mode)
- Secondary: #6459A7 (purple)
- Accent: #3DB1AC (teal)
- Danger: #EB3D63
- Success: #10B981
- Warning: #FFC107
- Background Dark: #0D1529 (deep navy, NOT black)
- Card Dark: #1A2550 (royal dark blue)
- Text Dark: #E1DEF5 (purple-white)

## Key Design Elements
- Glassmorphism: backdrop-blur(20px), semi-transparent backgrounds
- Glow effects on interactive elements
- Scan line effects on cards
- Orbital particles around logo
- Animated counters for KPI values
- Shimmer loading states
- Spring animations for transitions
- Sound effects (optional)
- Font: Tajawal (Arabic + Latin)
- Chart colors: ['#3DB1AC', '#6459A7', '#EB3D63', '#FFC107', '#273470']
